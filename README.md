# Book Rental Backend 📚

Spring Boot backend for the Online Book Rental System.

## 🧰 Tech Stack

- Java
- Spring Boot
- Maven
- H2 Database (in-memory)

## 🧪 Features

- REST APIs for book CRUD operations
- Connected with React frontend
- Dummy payment API

## ▶️ How to Run Backend

```bash
./mvnw spring-boot:run
